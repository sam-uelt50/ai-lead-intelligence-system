# Agency Lead Intelligence System 🎯

Signal-based lead intelligence for quality conversations. This system helps agencies identify and prioritize companies showing real growth signals, enabling meaningful outreach instead of spam.

## ✨ Features

- **Signal Detection**: Automatically detects growth signals (hiring, funding, tech changes)
- **Intelligent Scoring**: Scores companies based on intent, timing, and fit
- **Research Briefs**: Generates AI-powered research briefs for quality outreach
- **Priority Queue**: Prioritizes leads based on multiple factors
- **RESTful API**: Fully documented API for integration
- **Modern Stack**: FastAPI, MongoDB, Async Python
